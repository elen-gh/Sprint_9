# Sprint_9
# Проект автоматизации тестирования сервиса "Продуктовый помощник"

Данный проект содержит автоматизированные UI-тесты для веб-приложения "Продуктовый помощник". Тесты реализованы на Python с использованием Selenium WebDriver, фреймворка PyTest и шаблона проектирования Page Object. 

Инфраструктура проекта контейнеризирована с помощью Docker и Selenoid, а также настроен CI/CD пайплайн на базе GitHub Actions.

## 📝 Покрытые тест-сценарии

1. **Создание аккаунта (`test_signup.py`)**
   * Регистрация пользователя с заполнением всех обязательных полей.
   * Проверка успешного редиректа на страницу авторизации и отображения формы входа.

2. **Авторизация (`test_signin.py`)**
   * Вход в систему с валидными учетными данными.
   * Проверка перехода на главную страницу и появления кнопки «Выход» в навигационной панели.

3. **Создание рецепта (`test_recipes_create.py`)**
   * Авторизация, переход в конструктор рецептов, динамическое добавление ингредиентов из выпадающего списка и заполнение формы.
   * Проверка создания рецепта: отображение карточки на главной странице и валидация его названия.
